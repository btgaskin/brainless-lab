`:freeze_plasticity`, `:zero_recurrent`, `:clamp_target`, `:disable_vision` (cross-family
perturbations), plus the compartmental `:reset_dendrites`, `:no_soma_back`,
`:no_hillock_back`.

## Sweep config schema

```toml
[sweep]     # id, mode = one_at_a_time | factorial, seeds = [...], max_cells
[baseline]  # node, task, N, ticks — the canonical setup
[axes]      # "node.<p>" / "env.<p>" / "drive.<p>" / "task.<p>" / "ablation" / "seed" -> [values]
[analytics] # measures = ["sigma_mr", "spectral_radius", "liveness", "regime", ...]
```

See [Tooling](/tooling/) for the walkthrough and `sweep --list-axes <node> <task>` to
discover what's tunable.
