Everything here is **Experimental**. `available` means implemented and discoverable;
`integrated` additionally requires a standard-runtime path, a checked example or
configuration, and focused tests. Neither label establishes biological fidelity, task
efficacy, or a scientific result.

Begin with the [Core handbook](/core/architecture/) for the stable contracts these features
extend. Study evidence has a separate ladder; feature readiness never promotes an experiment.
