These features compose through the [Core embodiment contract](/core/embodiment/).
Integration does not imply biological or robotic fidelity.

`SectorVision` provides observer-relative, fixed-width banks for objects or conspecifics,
with explicit output gain and distance-response exponent.
`AntagonisticTurnActuator` maps left-turn, right-turn, and thrust outputs to a unicycle
command, so reservoir activity can vary both turn and speed. `ProximityExposure` is a world
relation: it can restore a regulated association variable from physical proximity
independently of what an agent can see. The small
`examples/shoal_forage_quickstart.jl` script exercises these components. It is a runtime
example, not a calibrated ecological model.
