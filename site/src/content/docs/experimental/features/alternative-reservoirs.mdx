SORN supplies online STDP, intrinsic plasticity, and normalisation. Compartmental reservoirs
instead use fixed-weight dynamics intended for evolution or another explicit preparation
procedure.

SORN criticality and evolved task behaviour remain unvalidated scientific claims. An
untrained compartmental reservoir is not a fair performance comparison with the
online-plastic canonical Falandays reservoir.
