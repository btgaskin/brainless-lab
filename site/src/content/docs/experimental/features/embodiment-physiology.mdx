The stable Core `Embodiment` contract composes identified physical parts. This feature
record tracks the current discoverable sensor, encoder, actuator, dynamics, and physiology
implementations, including optional regulated variables. Configuration round trips, the
object-world quickstart, and focused runtime tests exercise that composition.

Integration does not establish biological or robotic fidelity. Multiple regulated variables,
feedback, and absorbing failure are implemented mechanisms; successful multi-need
self-regulation requires a separately calibrated experiment.
