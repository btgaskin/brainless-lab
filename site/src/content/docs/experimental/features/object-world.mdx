`ObjectWorld` closes composed bodies through a small physical world with stable object
identity, resource arbitration, simplified appearance, fields, and typed effects. Both
direct `Ensemble` and `TaskSpec` examples exercise the path.

It models static circles in a simplified two-dimensional arena. It is not a general
rigid-body, optical, robotic, or ecological simulator.
