Development maps bounded scalar genotype values onto a fresh fixed component graph.
Evolutionary drivers evaluate compatible reservoir or composite genomes through declared
task and preparation contracts.

The component graph remains fixed, and one optimizer run does not characterize search
variance. A selected genome is development output until an untouched, appropriately powered
comparison confirms a scientific claim.
