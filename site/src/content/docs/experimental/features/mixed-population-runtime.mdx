The ensemble can group heterogeneous concrete agent types behind function barriers while
stable entity IDs retain world and recorder identity independently of group storage.

The current population is fixed. Birth, migration, replacement, lineage scheduling, and
variable-port evolutionary materialization are outside this feature's scope.
