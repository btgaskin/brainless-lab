The analysis layer reads recorded channels without changing the simulated trajectory. It
provides node- and agent-scale descriptors, windowed variants, and surrogate helpers through
one `SimResult`-based interface.

Tested computation does not establish construct validity. Criticality and information-flow
claims need estimator diagnostics, appropriate nulls, independent worlds, and declared
finite-size and multiplicity handling.
