These variants add state-dependent membrane noise or stochastic receptor input while
retaining the shared reservoir interface. The variant tour runs the canonical baseline,
Oosawa drive, and sensory-noise condition through one task-facing path.

They are not part of the authors-faithful `:falandays` baseline. Oosawa-inspired
spontaneous activity is a modeling interpretation, not validation of a physiological
mechanism.
