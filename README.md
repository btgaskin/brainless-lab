# BrainlessLab.jl

BrainlessLab.jl is an extensible Julia research-tinkering library for exploring neurons as nodes within a collective, with clean interfaces for reservoirs, bodies, media, tasks, drives, interventions, and evolution strategies. Later stages will add concrete node families, including the default Falandays node, along with environments, worlds, analysis, and visualization tools; the full README arrives at a later stage.
