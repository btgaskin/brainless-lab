# Documentation has moved

The BrainlessLab documentation now lives in the **Astro/Starlight docs-and-outputs site**
under [`site/`](../site/), published at <https://brainless-lab.pages.dev>. That site is the single
source of truth; the old plain-markdown `docs/*.md` pages have been retired.

Browse it online, or run it locally:

```bash
cd site
bun install
bun run dev        # http://localhost:4321
```
